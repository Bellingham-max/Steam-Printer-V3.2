# 复古蒸汽卡片机 (Steam-Printer V3.2)

一个具有复古蒸汽朋克风格的交互式卡片生成器，可以创建精美的复古风格卡片并保存为图片。

## ✨ 功能特性

- **多种纸张材质**：羊皮纸、牛皮纸、蓝图、旧报纸、黑卡
- **丰富字体选择**：西部、打字机、手写体、哥特、海报、代码等 6 种字体
- **自定义墨水颜色**：深褐、纯黑、血红、海军蓝、金漆、粉笔白
- **多样边框样式**：加厚相框、复古双线、虚线边框、无边框
- **印章装饰**：绝密、加急、通过、典藏等印章可选
- **交互式桌面**：卡片可拖拽移动，支持多卡片管理
- **图片导出**：支持单张卡片保存和桌面全景保存
- **精美动画**：卡片生成带有流畅的掉落动画效果

## 🚀 使用方法

1. 在文本框中输入您想要打印的文字内容
2. 选择您喜欢的纸张材质、字体、墨水颜色、边框样式和印章
3. 点击 **⚡ 启动印刷 (PRINT) ⚡** 按钮生成卡片
4. 卡片会出现在桌面上，您可以：
   - 拖拽卡片到任意位置
   - 鼠标悬停在卡片上，点击 💾 按钮保存单张卡片
   - 使用右下角工具栏：
     - 📷 保存桌面全景
     - 🗑️ 清空桌面

## 🎨 界面预览

应用采用复古蒸汽朋克风格设计，包含：
- 深色背景与噪点滤镜效果
- 黄铜色机械控制台

## 📦 技术栈

- **纯 HTML/CSS/JavaScript**：无需构建工具，直接运行
- **html2canvas**：用于将卡片和桌面导出为图片
- **Google Fonts**：提供多种复古字体支持

## 🌐 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge 等）

## 📝 使用说明

### 纸张材质选项
- **羊皮纸 (Parchment)**：经典米黄色羊皮纸效果
- **牛皮纸 (Kraft)**：温暖的棕色牛皮纸
- **蓝图 (Blueprint)**：深蓝色工程蓝图风格
- **旧报纸 (News)**：泛黄的旧报纸效果
- **黑卡 (Dark Card)**：深色背景配金色文字

### 字体选项
- **西部 (Western)**：粗犷的西部风格字体
- **打字机 (Typewriter)**：经典打字机等宽字体
- **手写体 (Script)**：优雅的手写风格
- **哥特 (Gothic)**：古典哥特式字体
- **海报 (Poster)**：醒目的海报字体
- **代码 (Tech)**：现代科技感等宽字体

### 印章选项
- **绝密 (TOP SECRET)**：红色印章
- **加急 (URGENT)**：红色印章
- **通过 (APPROVED)**：绿色印章
- **典藏 (VINTAGE)**：紫色印章

## 💡 提示

- 卡片支持多行文本，会自动换行
- 每次生成的卡片位置和角度略有随机，营造真实感
- 拖拽卡片时会有轻微的旋转效果
- 保存图片时建议使用高分辨率显示器以获得最佳效果
